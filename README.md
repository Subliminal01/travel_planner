# 🌍 Travel Planning & Experience Engine

An intelligent, dynamic travel itinerary generator built for PromptWars. This MVP not only creates personalized travel plans based on user preferences but features a **Real-Time Disruption Simulator** that instantly recalculates itineraries when unexpected events (like bad weather or budget cuts) occur.

## 🚀 Features
*   **Dynamic Itinerary Generation:** Tailored day-by-day plans based on destination, vibe, and budget.
*   **Real-Time Constraint Engine:** Instantly adapts to injected disruptions (e.g., swapping outdoor activities for indoor ones during a simulated thunderstorm).
*   **Modern UI:** Clean, responsive dashboard interface.

## 🛠️ Tech Stack
*   **Frontend:** React / Vite / Tailwind CSS
*   **Deployment:** Vercel

## 💻 How to Run Locally
1. Clone the repository:
   `git clone https://github.com/Subliminal01/travel_planner.git`
2. Install dependencies:
   `npm install`
3. Start the development server:
   `npm run dev`
